# Anki 导出格式

从重大盲区和高频薄弱主题生成闪卡。语言与用户对话一致。

## CSV 导入

Anki：`文件 → 导入 → 类型：基础 → 允许 HTML`

```csv
Front,Back,Tags
"volatile 能保证原子性吗？","不能。volatile 只保证可见性和禁止指令重排。","Java::并发 盲区"
```

### 字段规则

| 字段 | 规则 |
|------|------|
| Front | 单一问题；≤ 80 字符 |
| Back | ≤ 3 句；可选 `相关: path/to/file` |
| Tags | `领域::子领域` + `盲区` |

## 卡片数量

| 对话规模 | 卡片数 |
|----------|--------|
| < 5 问题 | 2–3 |
| 5–15 | 3–5 |
| > 15 | 5–8 |

## 检查清单

- [ ] 每张只测一个概念
- [ ] 正面独立可答
- [ ] 至少一张关联源码（若对话有触达）
- [ ] 语言与对话一致
